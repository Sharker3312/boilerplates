# Dashboards Overview

- `stack-overview.json`: resumen de salud del stack de observabilidad.
- `dermagroup-dev.json`: observabilidad de logs para DermaGroup en ambiente dev.
- `heyyou.json`: monitoreo de logs de la aplicación HeyYou.
- `lambda/todomercado-lambdas-dashboard.json`: monitoreo unificado de AWS Lambda en TodoMercado por `stage` y `lambda_name`.
